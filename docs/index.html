CensorCore

A lightweight JavaScript library for filtering explicit or inappropriate text.

Overview

CensorCore is a lightweight JavaScript library for detecting explicit or inappropriate text using a centralized JSON wordlist. It integrates with a single script tag and one function call.

Features

  • Zero configuration setup
  • Shared JSON wordlist loaded from CDN
  • Efficient whole‑word matching
  • Minimal and predictable API
  • No DOM modification or event interception
  • Full control retained by the integrating developer

Installation

<script src="https://cdn.jsdelivr.net/gh/DerrickRichard/CensorCore-Library/CensorCore.js"></script>

Usage

if (censor.isBlocked(text)) {
  alert("Message blocked: Inappropriate Content");
  return;
}

Wordlist Structure

{
  "profanity": ["word1", "word2"],
  "hate_speech": ["word3", "word4"],
  "harassment": ["word5", "word6"]
}

API Reference

censor.isBlocked(text: string): boolean

Design Philosophy

CensorCore avoids modifying the DOM and provides a predictable function for message filtering.
